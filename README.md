## UVbox

UV box for PCB manufacturing

### Architecture

```
UVbox ~
       \
       | --- structure - 3D models of the mechanical parts
       |
       | --- software  - software for control and measurement
       |     |
       |
       |
       \ --- hardware  - all the schematics
```
